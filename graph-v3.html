The circles represent entities and might themselves contain other entities. On the left, the info panel gives you information on the currently selected node.

Instructions:

  • One click: select a node
  • Two clicks: select and expand a node by showing the nodes it contains
  • Scroll up/down: Zoom in/out on the graph
  • Search for a particular node with the search field on top

It is advised to look on the Chrome browser
